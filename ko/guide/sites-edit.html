사이트 편집 개요 ​

VpAdmin의 사이트 편집 페이지는 웹사이트를 관리하고 편집하기 위한 강력하고 간소화된 인터페이스를 제공하여 콘텐츠 생성부터 배포까지 전체 게시 프로세스를 단순화합니다. 파일 트리 보기, 마크다운/코드 편집기 및 강력한 버전 제어 도구를 결합하여 사이트의 콘텐츠, 구조 및 배포 워크플로를 쉽게 미세 조정할 수 있습니다.

개요 ​

이 페이지는 세 개의 주요 섹션으로 나뉩니다.

  1. 왼쪽 사이드바: 프로젝트 파일 트리, 다양한 작업 및 프로젝트 관련 정보에 대한 액세스를 제공합니다.

  2. 크기 조정기: 왼쪽 사이드바의 너비를 조정하는 데 사용되는 드래그 가능한 분리 기호입니다.

  3. 기본 콘텐츠 영역: 편집기에 파일 콘텐츠를 표시하고, 여러 개의 열린 파일을 위한 탭과 프로젝트 관리 도구를 표시합니다.

왼쪽 사이드바 ​

왼쪽 사이드바는 프로젝트 탐색 및 관리를 위해 설계되었으며, 몇 가지 주요 구성 요소가 포함되어 있습니다.

헤더 토글 ​

  • 기능: 프로젝트 선택기, 새로 고침 버튼, 프로젝트 할당량 진행률을 포함하는 헤더 섹션의 가시성을 토글합니다.

  • 작업: 헤더 토글 아이콘을 클릭하면 헤더 가시성이 토글됩니다.

프로젝트 선택기 ​

  • 기능: 편집할 프로젝트를 선택합니다.

  • 작업: 드롭다운을 클릭하고 목록에서 프로젝트를 선택합니다.

프로젝트 할당량 진행률 ​

  • 기능: 현재 사이트 사용 할당량을 표시합니다. 표시되는 저장 용량은 사이트에 대해 선택한 가격 계획에 따라 다릅니다. 자세한 내용은 가격을 참조하십시오.

  • 프로젝트 할당량 진행률 스크린샷:

작업 표시줄 ​

  • 기능: 파일을 관리하고 사이트를 변경하는 데 사용되는 버튼 세트를 제공합니다.

  • 레이아웃: 작업 표시줄에는 다음 버튼이 포함되어 있습니다.

    • 복구 지점 표시: 프로젝트의 복구 지점에 액세스하고 관리하여 이전 버전으로 되돌리는 데 도움을 줍니다.
    • 모두 저장: 변경한 모든 내용을 저장합니다.
    • 모두 확장: 파일 트리의 모든 노드를 확장합니다.
    • 모두 축소: 파일 트리의 모든 노드를 축소합니다.
    • 모든 파일 표시/숨기기: 모든 파일 또는 사이트 콘텐츠와 관련된 파일만 표시하도록 전환합니다.
    • 프로젝트 다운로드: 사이트 파일이 포함된 zip 파일을 다운로드합니다.
    • 변경된 파일 표시/숨기기: 트리에서 변경된 파일의 표시를 전환합니다.
    • 기록 보기: 커밋 기록을 열어 이전 버전으로 되돌릴 수 있습니다.
    • 변경 사항 제출: 선택한 변경된 파일을 사이트의 새 버전으로 제출하지만, 수정 사항은 게시하지 않습니다.
    • 프로젝트 게시: 웹사이트를 게시하여 변경 사항을 도메인에서 라이브 상태로 만듭니다.

  • 작업 표시줄 스크린샷:

게시 진행률 표시줄 ​

  • 기능: 사이트 게시의 현재 진행률 또는 가장 최근 게시의 결과를 표시합니다.

  • 작업: "+" 버튼을 클릭하여 현재 진행률에 대한 세부 정보를 숨기거나 표시합니다. 현재 사이트가 게시되지 않는 경우 마지막 게시의 결과가 표시됩니다. 마지막 게시가 실패한 경우 오류 로그를 볼 수 있습니다.

파일 트리 ​

  • 기능: 사이트 파일 구조를 표시하여 편집할 파일을 탐색하고 선택할 수 있습니다.

  • 기능:

    • 탐색: 화살표 아이콘을 사용하여 폴더를 확장하고 축소합니다.
    • 파일 선택: 파일 이름을 클릭하여 편집기에서 파일을 엽니다.
    • 드래그 앤 드롭: 파일을 드래그 앤 드롭하여 사이트를 구성합니다. 로컬 파일을 드래그 앤 드롭하여 사이트에 업로드할 수도 있습니다.
    • 컨텍스트 메뉴: 컨텍스트 메뉴 버튼 위에 마우스를 가져가면 노드 관련 작업이 더 많이 제공되는 드롭다운 메뉴가 나타납니다.

변경된 파일 보기 ​

  • 기능: 상태 표시기와 함께 수정된 모든 파일을 표시합니다. 또한 변경 사항을 비교하고 수정 사항을 되돌릴 수 있습니다.

버전 기록 보기 ​

  • 기능: 버전 기록을 표시하여 프로젝트 버전을 탐색하고 사이트를 특정 버전으로 되돌릴 수 있습니다. 참고: 이 버전 이후에 이루어진 모든 변경 사항은 손실됩니다. 시스템은 현재 사이트 상태를 저장하기 위해 복구 지점을 자동으로 만듭니다. 또한 녹색 아이콘이 있는 버전은 게시되었고, 회색 아이콘이 있는 버전은 아직 온라인에 게시되지 않았습니다.

크기 조정기 ​

  • 기능: 왼쪽 사이드바를 왼쪽이나 오른쪽으로 드래그하여 너비를 조정할 수 있습니다.

기본 콘텐츠 영역 ​

기본 콘텐츠 영역은 다양한 파일 형식 및 프로젝트 세부 정보를 보고 편집하기 위한 인터페이스를 제공합니다.

탭 ​

  • 기능: 여러 개의 열린 파일을 탭으로 표시하여 쉽게 탐색할 수 있습니다.
  • 기능:

    • 탭 레이블: 파일 이름을 표시하고 점과 로딩 표시기를 사용하여 저장되지 않았거나 저장 중인 상태를 나타냅니다.

    • 마우스 오른쪽 버튼 클릭 메뉴: 탭을 마우스 오른쪽 버튼으로 클릭하여 닫기, 새로 고침과 같은 옵션에 액세스합니다.

파일 편집기 ​

  • 기능: 마크다운, 코드 및 이미지를 포함한 다양한 형식을 기반으로 파일 콘텐츠를 보고 편집할 수 있는 편집기를 제공합니다.

  • 기능:

    • 마크다운 편집기: 서식 있는 형식, 실시간 미리 보기 및 마크다운 문서 저장을 지원합니다.
    • 코드 편집기: 코드 파일의 구문 강조 및 편집을 지원합니다.
    • 이미지 뷰어: 이미지 파일을 표시합니다.
    • Diff 보기: 두 버전 간의 변경 사항을 표시합니다.

결론 ​

VpAdmin의 사이트 편집 페이지는 웹사이트 콘텐츠 관리를 위한 간소화된 환경을 제공하여 강력한 버전 제어 및 단순화된 게시 워크플로의 이점을 누리면서 프로젝트를 쉽게 탐색하고, 파일을 편집하고, 변경 사항을 효과적으로 배포할 수 있도록 해줍니다. 직관적인 디자인과 강력한 기능을 통해 VpAdmin을 통해 웹사이트를 원활하게 유지 관리하고 업데이트할 수 있습니다.
