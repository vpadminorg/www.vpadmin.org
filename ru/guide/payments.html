Обзор Платежей ​

Модуль управления платежами VpAdmin позволяет вам легко интегрироваться с удобными для авторов платформами, такими как Ko-fi, что позволяет вам управлять платежами и улучшать пользовательский опыт на вашем веб-сайте. Этот модуль предоставляет инструменты для настройки параметров оплаты, управления заказами и оптимизации процесса продажи цифрового контента или подписок.

Поддерживаемые Платформы ​

В настоящее время модуль управления платежами в основном поддерживает Ko-fi, платформу, популярную среди авторов для получения пожертвований и продажи цифровых продуктов. Поддержка других платформ может быть добавлена в будущих выпусках.

Ключевые Функции ​

  1. Конфигурация Платежей: Добавляйте и управляйте конфигурациями платежей для своего веб-сайта, позволяя вашему сайту интегрироваться с возможностями оплаты Ko-fi, завершать управление заказами и обрабатывать автоматические/ручные ответы.
  2. Улучшенные Кнопки Поддержки: Добавляйте на свой веб-сайт кнопки оплаты, которые запускают процессы оплаты Ko-fi. Вы можете разместить любое количество кнопок оплаты в любом месте содержимого вашего веб-сайта.
  3. Автоматические Ответы по Электронной Почте: Настройте автоматические ответы по электронной почте пользователям после успешной оплаты.
  4. Управление Заказами: Просматривайте и управляйте заказами на оплату, связанными с вашим веб-сайтом. Для получения более подробной информации, пожалуйста, смотрите Заказы.

Настройка Параметров Платежей ​

Выполните следующие шаги, чтобы настроить параметры оплаты для вашего веб-сайта:

  1. Введите Детали Конфигурации: Заполните необходимую информацию, включая:

    • Название Сайта: Выберите сайт, к которому будет применяться эта конфигурация платежей.

    • Название Платежа: Введите описательное название для этой конфигурации платежей. Названия платежей должны быть уникальными в пределах одного и того же сайта.

    • Платежная Платформа: Выберите "ko-fi" в качестве платежной платформы.

    • Токен Верификации: Этот токен будет использоваться для проверки веб-хуков Ko-fi. Получите Токен Верификации из своей учетной записи Ko-fi в разделе API -> Дополнительно, как показано на скриншоте ниже.

      Токен верификации Ko-fi

    • Тип Платежа: Выберите тип платежа. Варианты: Onetime, Membership, MembershipWithTier и shop (в настоящее время соответствуют типам платежей, доступным на Ko-fi).

    • Уровень Пользователя: Если тип платежа MembershipWithTier, установите название уровня в соответствии с тем, которое настроено в Ko-fi.

    • Автоответ: Включите или отключите автоматические ответы по электронной почте после успешной оплаты. Информация о вашем ответе также будет отображаться в деталях заказа для пользователей на вашем сайте.

    • Тема: Строка темы для автоматического письма.

    • Содержание: Содержание автоматического письма. Вы можете предоставить общедоступную ссылку на контент, который вы продаете, например, документ PDF.

      ПРИМЕЧАНИЕ

      VpAdmin не несет ответственности за хранение какого-либо вашего защищенного контента. Пожалуйста, не помещайте защищенный цифровой контент, лицензионные ключи и т.д. непосредственно в содержание электронной почты.

  2. Сохранить Конфигурацию: После ввода всей необходимой информации нажмите "Подтвердить", чтобы сохранить новую конфигурацию платежей.

Чтобы узнать больше об управлении заказами и принципах автоматических/ручных ответов, пожалуйста, обратитесь к Управлению Заказами.

Добавление Улучшенных Кнопок Поддержки ​

После настройки конфигурации платежей вы можете добавить улучшенные кнопки поддержки к содержимому вашего веб-сайта. Следуйте приведенным ниже примерам кода в зависимости от выбранного вами фреймворка Генератора Статических Сайтов (SSG):

<SupportButton
platform="ko-fi"
paymentType="donation"
text="Поддержать меня"
url="https://ko-fi.com/vpadmin/?hidefeed=true&widget=true&embed=true&preview=true"
name="Премиум Подписка"
:needLogin="true"
size="default"/>
{{< support-button
platform="ko-fi"
paymentType="donation"
text="Демо-кнопка Поддержки"
url="https://ko-fi.com/vpadmin/?hidefeed=true&widget=true&embed=true&preview=true"
name="Премиум Подписка"
needLogin="true"
size="default" >}}
{% include support-button.html
platform="ko-fi"
paymentType="donation"
text="Демо-кнопка Поддержки"
url="https://ko-fi.com/vpadmin/?hidefeed=true&widget=true&embed=true&preview=true"
name="Премиум Подписка"
needLogin="true"
size="default" %}

Объяснение Параметров Кода:

  • platform: Платежная платформа, в настоящее время поддерживает только "ko-fi".
  • paymentType: Тип платежа, в настоящее время поддерживает только "donation" (соответствует "Onetime" в конфигурации) или "subscription" (соответствует "Membership" и "MembershipWithTier" в конфигурации).
  • text: Текст, отображаемый на кнопке поддержки.
  • url: URL вашего профиля Ko-fi. Просто замените "https://ko-fi.com/vpadmin/" в URL на адрес вашей страницы Ko-fi.
  • name: Название конфигурации платежа.
  • needLogin: Нужно ли пользователю входить в систему перед совершением платежа.

    ПРИМЕЧАНИЕ

    Чтобы автоматически связывать заказы Ko-fi с пользователями на вашем веб-сайте, установите для этого параметра значение true.

  • size: Размер кнопки ("small", "default").

Посмотрите демонстрацию кнопки оплаты ниже:

<SupportButton
platform="ko-fi"
paymentType="donation"
text="Поддержать меня"
url="https://ko-fi.com/vpadmin/?hidefeed=true&widget=true&embed=true&preview=true"
name="Премиум Подписка"
:needLogin="false"
size="default"/>

Управление Заказами на Оплату ​

VpAdmin предоставляет панель управления для просмотра и управления заказами на оплату. Вы можете просматривать все заказы, связанные с вашим веб-сайтом, просматривать детали заказа и отслеживать платежи. Кроме того, VpAdmin также поддерживает автоматические/ручные ответы в соответствии с вашей конфигурацией. Для получения более подробной информации, пожалуйста, смотрите Заказы.

Заключение ​

С помощью модуля управления платежами VpAdmin вы можете легко интегрировать функциональность оплаты в свой веб-сайт, что позволит вам монетизировать свой контент и расширить свое присутствие в Интернете. Используя возможности Ko-fi и наши улучшенные кнопки поддержки, вы можете обеспечить бесперебойный и профессиональный опыт оплаты для ваших пользователей.
