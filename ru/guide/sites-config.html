Обзор конфигурации сайтов ​

Страница Конфигурация сайтов в VpAdmin позволяет вам управлять всеми своими веб-сайтами в одном месте. Здесь вы можете добавлять новые веб-сайты, редактировать существующие конфигурации сайтов, просматривать сведения о сайте и даже удалять веб-сайты. Она предоставляет централизованный интерфейс для управления и настройки вашего присутствия в Интернете.

Добавление нового сайта ​

Чтобы добавить новый веб-сайт:

  1. Нажмите «Добавить новый сайт»: Нажмите кнопку «+ Добавить новый сайт». Появится диалоговое окно.
  2. Подключиться к Git: Нажмите «Подключиться к GitHub» или «Подключиться к GitLab», чтобы авторизовать VpAdmin для доступа к вашему репозиторию.
  3. Настроить параметры сайта: Заполните необходимую информацию для вашего нового сайта, включая:
    • Название сайта: Введите описательное название для вашего сайта.
    • Тема: Выберите бесплатную тему для своего веб-сайта (например, VitePress).
    • Имя пользователя: Автоматически заполняется системой.
    • Адрес репозитория: Автоматически заполняется системой.
    • Админ-панель: Выберите «Да» или «Нет», чтобы включить или отключить панель администратора.
    • Домен: (Необязательно) Введите собственный домен или оставьте домен по умолчанию, если у вас нет своего собственного. Этот URL будет использоваться как домен обратного вызова для конфигураций OAuth. Если вам нужно добавить сторонний вход OAuth на свой веб-сайт, настройте правильный URL для своего веб-сайта. Пожалуйста, обратитесь к разделу OAuths для получения подробной информации.
  4. Нажмите «Подтвердить»: Сохраните новые конфигурации сайта, что инициирует первоначальное развертывание веб-сайта.

(См. аннотированный скриншот ниже для руководства)

Редактирование конфигурации сайта: ​

  1. Нажмите «Редактировать»: Нажмите значок редактирования рядом с сайтом, который вы хотите изменить.

  2. Изменить настройки: В диалоговом окне «Редактировать конфигурацию сайта» вы можете повторно подключиться к GitHub или GitLab и настроить параметры домена.

  3. Нажмите «Подтвердить»: Нажмите кнопку «Подтвердить», чтобы сохранить изменения, или «Отмена», чтобы отменить их.

Просмотр сведений о сайте: ​

  1. Нажмите «Подробности»: Нажмите значок подробностей, чтобы просмотреть подробные сведения о сайте.

Удаление сайта: ​

  1. Нажмите «Удалить»: Нажмите значок корзины рядом с сайтом, который вы хотите удалить.
  2. Подтвердить удаление: В диалоговом окне подтверждения подтвердите, что вы хотите удалить конфигурацию сайта.
    • Возможность удалить связанный удаленный репозиторий и веб-сайт.

    DANGER

    Это приведет к отключению вашего онлайн-сайта и его удалению.

Поиск сайта ​

Используйте фильтр в верхней части списка сайтов для поиска конкретного сайта.

Разбивка на страницы сайтов ​

Используйте функцию разбивки на страницы, чтобы найти определенный сайт, если сайтов слишком много.

Заключение ​

Страница Конфигурация сайтов в VpAdmin дает вам полный контроль над вашими веб-сайтами, позволяя быстро добавлять, редактировать, просматривать и эффективно управлять своим присутствием в Интернете.
