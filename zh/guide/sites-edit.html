站点编辑概述 ​

VpAdmin 中的站点编辑页面提供了一个强大且精简的界面，用于管理和编辑您的网站，简化了从内容创建到部署的整个发布过程。它结合了文件树视图、markdown/代码编辑器和强大的版本控制工具，使您能够轻松地微调站点的内容、结构和部署工作流程。

概述 ​

该页面分为三个主要部分：

  1. 左侧边栏： 提供对项目文件树、各种操作和项目相关信息的访问。

  2. 调整器： 一个可拖动的分隔符，用于调整左侧边栏的宽度。

  3. 主内容区域： 在编辑器中显示文件内容、不同打开文件的标签以及项目管理工具。

左侧边栏 ​

左侧边栏专为项目导航和管理而设计，包括几个关键组件：

头部切换 ​

  • 功能： 切换包含项目选择器、刷新按钮和项目配额进度的头部部分的可见性。

  • 操作： 点击头部切换图标可切换头部可见性。

项目选择器 ​

  • 功能： 选择您要编辑的项目。

  • 操作： 单击下拉列表，并从列表中选择项目。

项目配额进度 ​

  • 功能： 显示当前站点使用配额。显示的存储容量将根据您站点的选定定价方案而有所不同。有关更多详细信息，请参阅 价格。

  • 项目配额进度截图：

操作栏 ​

  • 功能： 提供一组按钮来管理文件并更改您的站点。

  • 布局： 操作栏包括以下按钮：

    • 显示恢复点： 允许您访问和管理项目的恢复点，帮助您恢复到早期版本。
    • 全部保存： 保存您所做的所有更改。
    • 全部展开： 展开文件树中的所有节点。
    • 全部折叠： 折叠文件树中的所有节点。
    • 显示/隐藏所有文件： 切换显示所有文件或仅显示与站点内容相关的文件。
    • 下载项目： 下载包含您的站点文件的 zip 文件。
    • 显示/隐藏已更改的文件： 切换显示树中已更改的文件。
    • 查看历史记录： 打开提交历史记录，允许您恢复到之前的版本。
    • 提交更改： 将您选择的已更改文件作为您站点的新版本提交，但不会发布修改。
    • 发布项目： 发布您的网站，使更改在您的域名上生效。

  • 操作栏截图：

发布进度条 ​

  • 功能： 显示您站点发布的当前进度或最近一次发布的结果。

  • 操作： 单击“+”按钮以隐藏/显示有关当前进度的详细信息。如果当前站点未发布，将显示上次发布的结果。如果上次发布失败，您可以查看错误日志。

文件树 ​

  • 功能： 显示站点文件结构，允许您导航和选择要编辑的文件。

  • 特点：

    • 导航： 使用箭头图标展开和折叠文件夹。
    • 文件选择： 单击文件名以在编辑器中打开该文件。
    • 拖放： 拖放文件以组织您的站点；您还可以拖放本地文件以上传到您的站点。
    • 上下文菜单： 将鼠标悬停在上下文菜单按钮上，以显示提供更多节点相关操作的下拉菜单。

已更改的文件视图 ​

  • 功能： 显示您修改的所有文件，并带有状态指示器。您还可以比较更改并还原修改。

版本历史视图 ​

  • 功能： 显示您的版本历史记录，允许您浏览项目的版本并将您的站点恢复到特定版本。注意：这将丢失在此版本之后所做的所有更改。系统将自动创建一个恢复点以保存当前站点状态。此外，带有绿色图标的版本已发布，而带有灰色图标的版本尚未在线发布。

调整器 ​

  • 功能： 允许您通过向左或向右拖动来调整左侧边栏的宽度。

主内容区域 ​

主内容区域提供了一个界面，用于查看和编辑不同的文件格式和项目详细信息。

标签页 ​

  • 功能： 将多个打开的文件显示为标签页，方便导航。
  • 特点：

    • 标签页标签： 显示文件名，并使用点和加载指示器指示未保存或正在保存状态。

    • 右键菜单： 右键单击标签页以访问诸如关闭、刷新之类的选项。

文件编辑器 ​

  • 功能： 提供一个编辑器，用于根据不同的格式（包括 markdown、代码和图像）查看和编辑文件内容。

  • 特点：

    • Markdown 编辑器： 支持丰富的格式、实时预览和保存 Markdown 文档。
    • 代码编辑器： 支持代码文件语法高亮和编辑。
    • 图像查看器： 显示图像文件。
    • 差异视图： 显示两个版本之间的更改。

结论 ​

VpAdmin 中的站点编辑页面为网站内容管理提供了一个精简的环境，使您能够轻松导航您的项目、编辑文件并有效地部署更改，同时受益于强大的版本控制和简化的发布工作流程。凭借其直观的设计和强大的功能，您可以通过 VpAdmin 无缝地维护和更新您的网站。
