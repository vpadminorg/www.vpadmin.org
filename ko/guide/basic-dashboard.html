기본 대시보드 개요 ​

VpAdmin 기본 대시보드는 웹사이트의 주요 지표와 성능 지표에 대한 빠른 개요를 제공하여 웹사이트의 성장과 활동을 한눈에 모니터링할 수 있도록 합니다.

주요 지표:

  • 총 사이트 수: 현재 VpAdmin 계정을 통해 관리되는 총 웹사이트 수를 표시합니다.
  • 총 사용자 수: 모든 사이트에 등록된 총 사용자 수를 표시합니다.
  • 총 주문 수: 결제 기능을 통합한 경우 모든 사이트에 접수된 총 주문 수를 나타냅니다.
  • 월별 수익: 현재 달에 웹사이트를 통해 발생한 총 수익을 표시합니다.
  • 평균 주문 금액: 주문당 평균 금액을 표시하여 고객 지출 행동에 대한 통찰력을 제공합니다.

추세 차트:

대시보드에는 시간 경과에 따른 웹사이트의 성능을 시각화하는 대화형 추세 차트가 포함되어 있습니다.

  • 사용자 증가 추세: 일정 기간 동안 등록된 사용자 증가를 추적합니다.
  • 주문 추세: 시간 경과에 따른 주문 수를 모니터링하여 추세와 패턴을 식별하는 데 도움을 줍니다.
  • 수익 추세: 시간 경과에 따른 월별 수익 변화를 확인합니다.

대화형 기능:

  • 데이터 새로 고침: "데이터 새로 고침" 버튼을 사용하여 표시된 모든 지표를 최신 정보로 업데이트합니다.
  • 더 많은 분석 미리보기: "더 많은 분석 데이터 미리보기" 버튼을 클릭하여 고급 분석 옵션을 탐색합니다.
  • 사이트로 이동: "사이트로 이동" 버튼을 클릭하여 웹사이트 관리 섹션으로 직접 이동합니다.

대시보드 스크린샷:

기본 대시보드는 사용자 친화적인 형식으로 필수적인 성능 통찰력을 제공하도록 설계되어 주요 지표를 모니터링하고 온라인 활동에 대한 정보에 입각한 결정을 내릴 수 있도록 합니다.
