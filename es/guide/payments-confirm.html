Resumen de Confirmaciones de Pagos ​

La página de Confirmaciones de Pagos en VpAdmin ofrece una vista integral de las confirmaciones de pago iniciadas desde tus sitios web. Esta página permite a los administradores monitorear las actividades de pago, gestionar los estados de los pedidos y responder a los usuarios, facilitando el manejo eficiente de las transacciones y mejorando la atención al usuario.

Resumen ​

La página de Confirmaciones de Pagos te permite:

  • Rastrear Confirmaciones de Pago: Ver una lista de todas las confirmaciones de pago recibidas desde tus sitios web.
  • Filtrar Confirmaciones de Pago: Filtrar las confirmaciones de pago por ID, nombre del proyecto, nombre del pago, correo electrónico del usuario, estado y hora.
  • Ver Detalles de Confirmación de Pago: Ver información detallada sobre cada pago, incluyendo información de pago asociada, correo electrónico del usuario, estado y contenido de la respuesta.
  • Actualizar Estado de Pago: Actualizar manualmente el estado de un pago y establecer un mensaje de respuesta para el usuario.
  • Actualizar Estados de Pago en Lote: Actualizar el estado de múltiples registros de pago a la vez.

Lista de Confirmaciones de Pago ​

La tabla principal muestra todos los registros de confirmación de pago con las siguientes columnas:

  • ID: El identificador único del registro de confirmación de pago.
  • Nombre del Proyecto: El nombre del proyecto asociado al pago.
  • Nombre del Pago: El nombre de la configuración de pago utilizada.
  • Correo Electrónico: La dirección de correo electrónico del usuario que inició el pago.
  • Estado: El estado actual del pago, con las siguientes opciones:
    • Pendiente: El sistema ha recibido la confirmación de pago y está esperando que el sistema haga coincidir el pago.
    • Éxito: El pago se ha completado y se ha hecho coincidir con el pedido de kofi.
    • Fallido: La coincidencia de la confirmación de pago falló
  • Hora: La hora de creación del registro de confirmación de pago.
  • Operaciones: Proporciona opciones para gestionar cada registro de confirmación de pago.

Operaciones Disponibles ​

  • Ver Detalles: Haz clic en el icono "Ver" para ver los detalles de una confirmación de pago.
  • El cuadro de diálogo de detalles proporciona la siguiente información:
* **Actualizar Estado:** Haz clic en el icono "Editar" para modificar el estado de una confirmación de pago y añadir un mensaje de respuesta. * Esto abre un cuadro de diálogo que te permite actualizar el estado y proporcionar respuestas personalizadas al usuario.

Flujo de Trabajo Clave ​

Coincidencia y Respuesta Automática de Pagos ​

  1. Usuario Inicia Pago: Cuando un usuario realiza un pago utilizando un Botón de Soporte en tu sitio web, VpAdmin registra la confirmación de pago del usuario en la página "Confirmaciones de Pagos".
  2. Procesamiento de Webhook: Si un pago de Ko-fi es exitoso y coincide con la configuración en la página de Configuración de Pago, VpAdmin intentará hacer coincidir automáticamente el pedido con el pago.
    • Si se encuentra una configuración de pago coincidente, VpAdmin cambiará automáticamente el estado del pago a 'Éxito' y proporcionará una respuesta automática por correo electrónico.
    • Si no se encuentra una configuración de pago coincidente, el registro de pago se marcará con el estado "Pendiente".
  3. Acción Manual del Administrador Requerida: Para los registros de confirmación de pago que no tienen un pedido coincidente y tienen el estado pendiente, es posible que los administradores deban cambiar manualmente el estado de esos registros de pago y enviar una respuesta.

Para obtener más información sobre la gestión de pedidos y los principios detrás de las respuestas automáticas/manuales, consulta Gestión de Pedidos.

Experiencia del Usuario ​

Los usuarios pueden ver el estado y la respuesta del administrador en la sección de Pagos del Centro de Usuarios.

Conclusión ​

La página Confirmaciones de Pagos es un componente vital para la gestión de confirmaciones de pago dentro de VpAdmin. Al proporcionar herramientas para el seguimiento, la actualización y la respuesta a los pagos, VpAdmin garantiza la transparencia y una experiencia de pago fluida tanto para administradores como para usuarios.
