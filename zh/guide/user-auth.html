用户身份验证 - 登录、注册等 ​

VpAdmin 中的用户身份验证系统为用户提供了一种安全便捷的方式来访问您的网站及其功能。它包括标准电子邮件/密码登录、社交登录和密码恢复选项，从而创建无缝的用户体验。

主要功能 ​

标准登录 ​

用户可以使用他们的电子邮件地址和密码登录，并提供验证码以确保安全。

社交登录 ​

VpAdmin 支持各种第三方身份验证平台，允许用户使用他们现有的帐户登录。支持的平台包括：

  • Github
  • Gitlab
  • Google
  • Facebook

注册 ​

新用户可以使用他们的电子邮件、密码和验证码来注册创建一个新帐户。

密码恢复 ​

  • 电子邮件代码验证： 用户可以通过输入他们的电子邮件地址并完成验证码来启动密码恢复。包含验证码的电子邮件将被发送到该地址。
  • 电子邮件代码验证： 输入电子邮件中的代码。
  • 设置新密码： 用户可以设置新密码。

优势 ​

  • 便捷登录： 用户可以使用他们现有的社交媒体帐户或标准的电子邮件和密码快速登录。
  • 安全访问： VpAdmin 确保用户帐户的安全并防止未经授权的访问。
  • 增强用户体验： 在登录、注册和密码恢复期间提供流畅无忧的用户体验。
  • 可定制： 登录用户界面可以配置为匹配您的品牌和设计美学。

通过提供简化的身份验证流程，您可以确保用户以安全、简单和便捷的方式访问您的平台。
